Usage Notes Get Started Flight Crew Support

DISCLAIMER: I made this app as a hobby and can not guarantee it will work perfectly in all circumstances.

This app is intended to support flight crew service members with aspects of their job. It features a fuel burn calculator and a METAR scraper/decoder. Read below for instructions on their use. Switch tabs to get started.

Fuel Burn Calculator Documentation

The fuel burn calculator is based on specifications determined by consulting with an active duty UH-60 Crew Chief. As a result, it may not be usable with other aircraft.

To begin, create a fuel tank and add a fuel amount to it. If you wish to include this tank in fuel burn rate calculations, select its checkmark icon. You can untrack it in the same way, and can also delete it completely by selecting its close icon. To begin a fuel check, hit the 'Start Fuel Check' button. A timer will start, and count up each second until 'Stop Fuel Check' is pressed.

Fuel burn rate is determined by taking the tracked fuel amount when the check is started and subtracting the tracked fuel when it is stopped, then dividing that by the time elapsed during the check. Time to burnout is then determined by taking the total fuel remaining (including fuel in untracked tanks) divided by the burn rate, and adding that time to current local time.

METAR Scraper Documentation

The METAR scraper uses an open source API to grab METAR weather reports from the web.

If you know the International Civil Aviation (ICAO) identification code for a specific airport or airbase, enter it in the search bar and select the 'Use ICAO Ident' button. Otherwise, by selecting the 'Use Current Device Location' button the app can also request your current coordinates (your device must allow location services for this) and generate the report for the nearest airport to those coordinates.
